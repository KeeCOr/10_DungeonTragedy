1. 게임 개요

턴제 카드 + 보드 전술 게임. 1명의 플레이어가 AI 아군 2~4명과 함께 3x5 그리드 위에서 3페이즈 드래곤 보스에 맞서 싸웁니다.

핵심 루프: 시작 화면 → 매치 시작 → 턴 반복(이동/공격/카드) → 매치 종료(스코어보드) → 3매치 후 최종 결과
  • 3매치 토너먼트 — 매치별 점수 합산, 최종 승자 결정
  • 히든 미션 — 각 플레이어에게 숨겨진 필수/선택 미션
  • 배신 요소 — 오크 종족은 적대적 미션 수행 가능

2. 보드 & 배치

  • 3행 × 5열 CSS Grid 보드
  • 드래곤: 맵 밖(off-grid)에 위치, 셀 점유 안 함
  • 공격존: 행 0 (상단행)에서만 드래곤 공격 가능
  • 시작 위치: 전원 행 2 (하단행)에서 시작
  • 보물 드롭: 드래곤 HP 감소 시 보드 위에 생성

6. 미션 시스템

플레이어당 필수 1 + 선택 1 미션이 숨겨진 상태로 배정됩니다.

점수 계산

  • 미션 완수 점수 합산
  • 마무리 보너스: 드래곤에 최종 타격 시 +3점
  • 생존 보너스: 매치 종료 시 생존 시 +1점
  • 3매치 합산 → 최고점 승리
